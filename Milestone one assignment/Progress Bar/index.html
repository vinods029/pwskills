Scroll Progress Bar

Scroll Down to See the Progress Bar Effect

This progress bar is an indicator to show how far the page is scrolled.

The progress bar effect will reverse on the scroll back.

This Scroll Progress Bar is responsive! It can work on different screen sizes.
